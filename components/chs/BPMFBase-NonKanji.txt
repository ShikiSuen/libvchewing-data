#==================================
# 威注音汉字音韵库格式，全文除了汉字与注音以外必须是半角英文数字。
# -------------------
# 格式顺序是：单字符号＆注音＆汉语拼音与声调数字＆注音对应美规键盘按键＆编码。
# 编码统一使用 UTF8 万国码；注音的轻声记号一律写在右侧。
# 每一行的每一项之间必须用单个英文半角空格来隔开，行尾不要留空格。
# 汉语拼音可借由其他脚本从注音换算，但注音一定要标注正确。
# 换算时请使用 LibreOffice，因为 Excel 会把 jun1 这样的数字永久转换成日期、导致数据损毁。
# 汉语拼音的音调必须是数字，「12345」分别对应「一二三四轻」这五种声调。
# 注释必须在单独的行内、且由西文井号字符开行头。
# ------------------
# 非汉字符号专用音韵库。
